# 周期ノート 🌸

生理周期を記録・予測するシンプルなウェブアプリです。  
データはすべてお使いの端末内にのみ保存されます（サーバーへの送信なし）。

## つかいかた

### ブラウザでそのまま使う

👉 **[www.azimicat.dev/cycle](http://www.azimicat.dev/cycle)**

Safari / Chrome でアクセスするだけで使えます。

---

### スマホのホーム画面に追加する（アプリとして使う）

アドレスバーが消え、アプリのように起動できます。

**iPhone (Safari)**
1. 上のURLをSafariで開く
2. 画面下の共有ボタン（□↑）をタップ
3.「ホーム画面に追加」を選ぶ

**Android (Chrome)**
1. 上のURLをChromeで開く
2. 画面右上のメニュー（⋮）をタップ
3.「ホーム画面に追加」またはインストールバナーをタップ

---

### ローカルで使う

インターネット接続なしで使いたい場合は、`src/周期ノート.html` を1ファイルダウンロードしてブラウザで開くだけです。

```
# リポジトリをクローンする場合
git clone https://github.com/azimicat/cycle.git
```

その後 `src/周期ノート.html` をダブルクリックして開くか、ブラウザにドラッグ＆ドロップしてください。  
※ 初回はReact等の読み込みにインターネットが必要です。

---

## 機能

| 機能 | 内容 |
|---|---|
| カレンダー | 生理日・予測日・排卵日・妊娠しやすい日を表示 |
| 記録 | 開始日・終了日・メモを記録・編集・削除 |
| 統計 | 平均周期・生理期間・周期の推移グラフ |
| 予測 | 次回の生理日・排卵日を自動計算 |
| バックアップ | JSONファイルでエクスポート・インポート |

## プライバシー

すべてのデータはブラウザの `localStorage` に保存されます。  
外部サーバーへの通信は一切ありません。
